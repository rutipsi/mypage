## üîß environment variables
To add analytics to your site, you can use Umami and Clarity. you will need to add the following environment variables to your `.env` file:

```plaintext
# Umami Analytics
UMAMI_WEBSITE_ID=your_umami_website_id
UMAMI_TRACKING_URL=your_umami_tracking_url

# Clarity Analytics
CLARITY_TRACKING_ID=your_clarity_tracking_id
```

if you don't want to use analytics, you can remove the analytics code from the [src/components/seo/base-head.astro](https://github.com/rutip/mypage/blob/main/src/components/seo/base-head.astro) file and remove the umami and clarity scripts.
